```java
subString(start, end) - Gets the subString in this interval. If the interval is not correct throws StringIndexOutOfBoundsExceptionn.

subString(start) - Gets the subString from the start index. If the interval is not correct throws StringIndexOutOfBoundsExceptionn.

replace(value1, value2) - Replaces all value1 occurrences with value2 if value1 exists in the string.

replaceAll(regex, value) - Replaces the values from the regex with a given value.

length() - Returns the length of the string.

contains(value) - Checks if the string contains the given value.

charAt(index) - Returns the character on the given index. If the index is invalid throws StringIndexOutOfBoundsException.

equals(String) - Returns true or false depending if the strings are equal.

equalsIgnoreCase(String) - Same as equals but ignores the Lower and Upper case letters.

compareTo(String) - Returns the difference in unicode between 2 strings.

compareToIgnoreCase(String) - Same as compareTo but ignores the Lower and Upper case letters.

concat(String) - Concatenates two strings. It returns new string.

indexOf(String) - If the string is contained, it returns the begin index of the string. It returns -1 if not found.

lastIndexOf(String)  - Returns the last occurrence of the string.

toLowerCase() - Returns all the letters of the string in lower case.

toUpperCase() - Returns all the letters of the string in upper case.

trim() - Removes the spaces in the beginning and the end of the string.

split(symbol) - Splits the words by a given symbol and adds them to array.

join(symbol) - Takes array of chars and joins the words with the given symbol and puts them into string.
```
